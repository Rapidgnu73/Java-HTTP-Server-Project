Project Details

HTTP Parsing

Parses request line, headers (Content-Length, Content-Type) and reads POST bodies accurately.

Routing

Simple path-based routing for '/', '/about', '/info', '/forms' and static resource paths.

Static Assets

Serves files from a 'www' folder using correct MIME types and binary-safe streaming for images.

Concurrency

Fixed thread pool (ExecutorService) handles many clients without blocking the main accept loop.

Logging

Access logs with timestamps, client IPs, and requested resources are appended to server_logs.txt.

Data Storage

Form submissions are parsed and persisted to a CSV file with enforced field ordering.
